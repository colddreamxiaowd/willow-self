# 国策模板 Brainstorm 组件

## 概述

PolicyBrainstorm 是一个基于 RSIP (递归稳态迭代协议) 方法论的国策设计辅助工具。它将传统的"国策树模板"转变为更直观、更便利的 Brainstorm 模式，帮助用户从零开始构建自己的国策树。

## 核心特性

### 🎯 设计原则引导
- 顶部轮播展示 RSIP 的8大设计原则
- 帮助用户理解"零敲牛皮糖"、"农村包围城市"等核心理念

### 📚 分类浏览
- **基础国策**：饭后洗碗、回家洗澡、沙发禁机等
- **睡眠作息**：先发制人、夜幕降临、电子宵禁等
- **数字健康**：卧室禁机、应用限额、通知清理等
- **晨间仪式**：光线唤醒、晨间运动等
- **专注工作**：神圣座位、番茄工作法、手机隔离等
- **晚间复盘**：赢麻了、晚间回顾等
- **应急机制**：水密隔舱、侦查任务等

### 🔍 智能筛选
- 按分类筛选
- 按难度筛选（简单/中等/较难）
- 关键词搜索（名称、描述、标签）

### 👍 推荐系统
- 标记"👍 推荐"的国策适合新手入门
- 基于 RSIP 原文作者的实际使用经验

### 📖 详细说明
每个国策包含：
- 💡 核心思路
- 📝 具体规则
- ✅ 成功标准
- ⚠️ 常见陷阱
- 🚀 升级路径（部分国策）
- 🎯 适用场景

### 🛒 国策购物车
- 点击国策卡片添加到"已选列表"
- 支持拖拽排序
- 一键生成国策树

## 使用方法

1. **浏览国策**：通过分类标签或搜索找到感兴趣的国策
2. **查看详情**：点击"查看详情"了解国策的完整信息
3. **添加到列表**：点击国策卡片或详情页的"添加"按钮
4. **调整顺序**：在底部已选面板中调整国策顺序
5. **生成国策树**：点击"生成国策树"按钮（待实现）

## RSIP 方法论核心原则

1. **零敲牛皮糖**：从鸡毛蒜皮的小国策开始，能拆就拆
2. **农村包围城市**：先解决简单问题，让难题自然变简单
3. **下必为例**：允许一次 = 永远允许，慎重设定边界
4. **有效干预节点**：在自由意志可干预的位置设置规则
5. **胜于易胜**：永远不要奢求一蹴而就，用小确胜积累
6. **穿越牛熊**：国策要在最糟糕的一天也能轻松存活
7. **杠杆思维**：四两拨千斤，用最小成本撬动最大改变
8. **递归回溯**：失败了就悔棋重来，直到找到最优路径
