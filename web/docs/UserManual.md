# PolicyTree 全功能修炼手册

欢迎使用 PolicyTree！本手册旨在帮助你从一名普通用户进化为“意志架构师”。我们将深入探讨如何利用这款工具的每一个角落来重塑你的自律生活。

---

## 1. 意志编织：编辑器高级技巧

### 1.1 节点管理
- **双击空白处**：创建新的根国策（意志的起点）。
- **右键菜单**：
    - **添加子国策**：建立依赖关系。子国策必须在父国策点亮后才能激活。
    - **复制国策**：快速复用成熟的自律逻辑。
    - **删除国策**：注意！删除父国策将导致其下的所有子树一并消失。
- **编辑面板**：点击节点可修改描述、阶层（Tier）以及**强化配置**。

### 1.2 级联逻辑 (Cascading Logic)
PolicyTree 模拟了真实的系统脆弱性：
- **激活条件**：你不能在没有根基的情况下点亮高级目标。
- **熄灭惩罚**：如果父国策（如“早起”）被熄灭，所有依赖它的子国策（如“晨读”、“冥想”）将自动变为不可用状态。这提醒你核心意志的重要性。我们想要使用可以被看见的节点崩塌（损失厌恶心理）重新审视整个习惯的逻辑。

### 1.3 快捷键秘籍
为了极致的流畅感，建议熟练掌握：
- `Ctrl + S`：保存当前进度到临时工作区。
- `Ctrl + C / V`：节点复制与粘贴。
- `Ctrl + Z`：撤销误操作。
- `Delete`：快速修剪意志树。
- **技巧**：在侧边栏【设置】中开启“显示快捷键提示”，可在屏幕下方看到动态引导。

## 2. 能量维持：执行与反馈

### 2.1 【执行终端】每日打卡
点击侧边栏（或书籍书签）进入打卡模式：
- **即时点亮**：完成任务后在打卡界面勾选，对应的国策树节点将同步点亮。
- **仪式感**：点亮动作伴随着视觉反馈，强化大脑的多巴胺回路。

### 2.2 【心理观测站】日志与分析
- **心情日志**：记录每日的心情波动。意志力受情绪影响极大，通过日志你可以发现自己的“低谷期”规律。
- **稳态分析**：查看你的意志网络健康度。系统会分析哪些国策经常“断裂”，哪些是你的“核心支柱”。

### 2.3 神圣座位 (Sacred Seat)
这是减少决策内耗的关键技巧：
- **空间绑定**：在侧边栏管理你的神圣座位。
- **原则**：例如将“书桌”绑定为“绝对禁止刷手机”的空间。通过环境暗示，让自律变成肌肉记忆。

## 3. 意志实验室：头脑风暴

当你不知道该如何开始时，进入【意志实验室】：
- **灵感激发**：利用内置的模板进行头脑风暴。
- **一键导入**：在实验室中生成的方案可以一键转化为正式的国策树节点。注意导入的节点为独立节点，需要手动添加到主树中。

---

## 4. 方案管理：国策档案库

### 4.1 多存档机制
系统提供 10 个独立的存档槽位：
- **存档 1 (迁移存档)**：系统会自动将你旧版的数据迁移至此。
- **自定义命名**：为不同的人生阶段（如“考试周”、“假期模式”）建立不同的意志树。
本来的目的是为了在目前没有后端的情况下，用户可以更好保存自己的国策树。

### 4.2 透视预览 (Peek Mode)
**这是最高级的使用技巧**：
- 在档案库中点击存档旁的 **👁️ (预览)** 图标。
- 你可以暂时看到该存档的完整逻辑，而不会覆盖你当前的进度。
- 适用于在不同人生方案之间进行快速对比。

---

## 5. 极致沉浸：禅修模式

点击页面右下角的**悬浮球（Maximize 图标）**：
- 进入全屏模式，隐藏侧边栏、顶栏和所有浮窗。
- **适用场景**：每周一次的深度人生复盘，或每天清晨的意志规划。
- 按 `Esc` 或再次点击悬浮球即可退出。

---

## 6. RSIP 协议深度解析

**RSIP (Recursive Steady-state Iteration Protocol)**：
- **递归 (Recursive)**：意志是可以无限细分的逻辑链。
- **稳态 (Steady-state)**：系统追求的是长期的平衡，而非短期的爆发。
- **迭代 (Iteration)**：允许失败，但必须通过档案库记录失败的原因并快速迭代出 V2.0 版本的自己。
