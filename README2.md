## 🎮 什么是国策树？

玩过《钢铁雄心4》吗？里面的**国策树**让玩家通过"点亮国策"来发展国家。

**PolicyTree 把这个机制搬到了现实生活中。**

想象一下：
- 每个目标都是树上的一个节点
- 完成一个目标，就能"点亮"它 ✨
- 点亮后解锁新的目标
- 看着整棵树逐渐被点亮，超有成就感！

**但 PolicyTree 不只是游戏化，它背后有硬核的科学机制帮你真正完成目标。**

### ✅ 国策树的三个秘密武器

**🌳 1. 可视化进度**

看着树上的节点一个个被点亮，就像游戏升级一样爽！

这种**即时反馈**让大脑持续获得奖励，更容易坚持。

**🚪 2. 水密舱保护**

今天没完成"读paper"？

没关系！系统会自动"关闭舱门"，保护你的其他目标不受影响。

**不会因为一次失败就全盘崩溃。**

**🪑 3. 神圣座位**

这是 PolicyTree 的秘密武器：

把**书桌前的椅子**变成"学习开关"。坐上去，身体就知道该学习了。

就像巴甫洛夫的狗听到铃声会流口水，
你的大脑会把"座位"和"学习"自动关联。

**不需要意志力，就像进厕所就想上厕所一样自然。**

## 🎯 核心机制（简单版）

### 国策树 = 把大目标拆成可执行的小步骤

```
        📚 晚间专注学习（终极目标）
           │
    ┌──────┼──────┐
    ▼      ▼      ▼
 🪑坐在   📵放下   📖开始
  书桌前   手机    读paper
    │      │      │
    ▼      ▼      ▼
  具体    具体    具体
  行动    行动    具体任务
```

**关键：每个国策都必须是具体的、可执行的行动。**

❌ "好好学习"（太抽象）
✅ "坐在书桌前"（具体行动）

---

### 水密舱 = 失败保护机制

```
┌─────────┐     ┌─────────┐     ┌─────────┐
│ 坐在    │────→│ 放下    │────→│ 开始    │
│ 书桌前  │     │ 手机    │🔒   │ 读paper │
│ ✅完成  │     │ ❌失败  │     │ 🔄保护  │
└─────────┘     └────┬────┘     └─────────┘
                     │
                     ▼
              自动关闭舱门
              "放下手机"失败
              不影响其他国策
```

**一次失败不会导致全盘放弃。**

---

### 神圣座位 = 环境触发器

```
坐在书桌前的椅子上
        ↓
大脑自动进入学习模式
        ↓
不需要意志力
不需要自我说服
身体自己就知道该做什么
```

**让环境帮你，而不是对抗环境。**

## 📖 实战示例：创建你的第一棵国策树

### 场景：晚上想专注学习，但总是玩手机

**Step 1：创建根国策**

```
名称：晚间深度学习
描述：每天晚饭后专注学习2小时
```

**Step 2：添加子国策（按执行顺序）**

```
子国策1：物理隔离手机
- 描述：把手机放在客厅充电
- 阻力：3（有点难，但能做到）

子国策2：激活神圣座位
- 描述：坐在书桌前的椅子上
- 阻力：1（很容易）

子国策3：打开学习材料
- 描述：翻开书本或打开paper
- 阻力：2（需要一点启动能量）

子国策4：开始番茄钟
- 描述：专注25分钟，休息5分钟
- 阻力：4（需要持续专注）
```

**Step 3：设置神圣座位**

```
座位名称：书桌前的椅子
绑定行为：阅读、写作、思考
违规惩罚：离开座位就记录一次违规
```

**Step 4：开始执行！**

```
晚上7点：
1. 把手机放客厅（点亮"物理隔离手机"）
2. 坐在书桌前（点亮"激活神圣座位"）
3. 翻开书本（点亮"打开学习材料"）
4. 开始25分钟专注（点亮"开始番茄钟"）
```

## 💡 国策设计原则

### ✅ 好的国策

- **具体可执行**："坐在书桌前" ✅
- **阻力适中**：不要太难（阻力8-10），也不要太简单（阻力1-2）
- **时间明确**："晚饭后立即执行"
- **结果可验证**：能清楚知道是否完成

### ❌ 不好的国策

- **太抽象**："好好学习" ❌
- **阻力过高**："每天学习12小时" ❌
- **时间模糊**："有空的时候" ❌
- **无法验证**："更自律" ❌

## 💪 使用技巧

**从小开始**

不要一上来就是"每天学习12小时"。

从"每天坐在神圣座位5分钟"开始。

**允许失败**

**找到你的黄金时间**

## 理论基础

本项目基于知乎文章《自制力问题的数学物理方法》，作者 edmond。

文章链接：https://www.zhihu.com/question/19888447/answer/1930799480401293785

### 核心思想

> "自制力问题，未必只能是一个心理学或生理学问题，或许也可以是一个能用数学、物理学去解决的工程问题。"

### 两代自控技术

**第一代：CTDP (链式时延协议)**

- **神圣座位**：物理位置 = 行为约束
- **预约信号**：提前设置行为触发器
- **链式触发**：完成一个国策自动激活下一个

**第二代：RSIP (递归稳态迭代协议)**

- **稳态分析**：评估当前行为模式的稳定性
- **水密舱机制**：失败时自动隔离，防止连锁崩溃
- **递归回溯**：系统找到最优干预路径

### 数学模型

**双曲贴现模型**

人在面临行为选择时，对某个选项的真实倾向可以分解为：

```
I = ∫₀^∞ V(τ) · W(τ) dτ
```

- **V(τ)**：未来价值函数
- **W(τ)**：权重贴现函数

这就是为什么我们更倾向于即时满足——因为 W(0) >> W(未来)。

作者 edmond 从小被严重 ADHD 和自控问题困扰，花费了从小学到博士的十几年光阴，完成了从"长期挣扎却无法专注一小时"到"无 DDL 下，连续数月在家全天专注自习"的蜕变。

> "我希望把它们作为礼物，送给那些和曾经的我一样，被 ADHD 和自控障碍折磨的人们。"
