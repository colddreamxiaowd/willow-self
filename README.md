# PolicyTree: 将意志编织成不可撼动的逻辑结构

> "可视化你的决心，让每一次微小的自律都成为通往宏大目标的基石。"

PolicyTree 是一款专为**追求卓越生活的个人**设计的意志力管理工具。基于 **RSIP (Recursive Steady-state Iteration Protocol)** 第二代自控技术，它将你的生活目标转化为如同策略游戏般的“国策树”，让自律变得具象、逻辑且充满成就感。

---

## 🌟 核心魅力

### 1. 意志的物理化 (Logic of Will)
告别碎片化的待办清单。将模糊的目标解构为具象的树状国策，每一项进步都清晰可见，逻辑严密，有效对抗决策疲劳。

### 2. RSIP 协议支持
内置专业级自控协议，支持：
- **即时点亮**：视觉化的反馈循环，点燃你的执行欲望。
- **强化系统**：记录每一项习惯的进化深度，见证从 0 到 1 的蜕变。
- **级联逻辑**：模拟真实系统，核心意志的动摇将自动预警关联目标。

### 3. 多维自控空间
- **【意志实验室】**：头脑风暴，探索最适合你的自律路径。
- **【执行终端】**：每日打卡，将意志转化为肌肉记忆。
- **【心理观测站】**：心情日志与稳态分析，实时监控你的心理状态。

### 4. 极致交互体验
- **禅修模式**：一键全屏，隐藏所有干扰，回归纯净的规划空间。
- **国策档案库**：10 个独立存档位，支持**即时预览**，无损尝试多种人生方案。

---

## 🚀 快速启动

只需简单三步，开启你的意志进化之旅：

1. **进入目录**：
   ```bash
   cd web
   ```
2. **安装依赖**：
   ```bash
   npm install
   ```
3. **开启应用**：
   ```bash
   npm start
   ```
   *应用启动后，浏览器将自动打开 http://localhost:5173*
